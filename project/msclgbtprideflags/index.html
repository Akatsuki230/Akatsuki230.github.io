LGBT Pride Flags

This mod adds LGBT pride flags to My Summer Car. It is a mod for the game My Summer Car by Amistech Games.

Flags Added:

  • Lesbian Flag
  • Gay Flag
  • Bi Flag
  • Trans Flag
  • Queer Flag
  • Intersex Flag
  • Custom flag creator!

Custom Flag Creator:

The custom flag creator allows you to create a custom flag from the following presets:

  • 2 stripes
  • 3 stripes
  • 3 stripes, middle is bigger
  • 4 stripes
  • 5 stripes
  • 6 stripes

These flags have fully customizable colors.

It also has the ability to save the flags and duplicate the flags.
