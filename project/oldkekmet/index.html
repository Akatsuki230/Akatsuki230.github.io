Old Kekmet Mod

This mod is deprecated and therefore is no longer updated. It is also no longer available for download.

This mod modifies/adds the following things to the Kekmet:

  • Old Engine Sounds (Default enabled)
  • Old Engine Starting Sounds (Default enabled)
  • Disable Dashboard
  • Disable Forklift Arm
  • Make Forklift White/Any other colour
  • Old Starting System (Default enabled)
  • Disable Radio
